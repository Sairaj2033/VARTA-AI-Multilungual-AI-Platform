## 📖 About

**Varta AI** is an open-source multilingual news platform that delivers AI-inspired news experiences without relying on real-time translation APIs.

Instead of making API requests during runtime, multilingual news articles are pre-generated and stored locally, allowing users to experience instant translations, smooth animations, and a responsive interface with zero latency.

---

## 🚀 Features

- 🧠 AI-inspired news experience
- 🌍 Multilingual support
- ⚡ Instant language switching
- 🎭 AI typing & shimmer animations
- 📂 Local JSON-powered content
- 📱 Fully responsive interface
- 🚀 Fast deployment on Render
- 🔒 No runtime translation API dependency

## 🎯 Why Varta AI?

Many existing news platforms:

- Primarily support English content
- Depend on costly real-time translation APIs
- Introduce noticeable translation delays
- Offer limited user interaction

Varta AI addresses these challenges by delivering multilingual news instantly through pre-generated content and AI-inspired user interactions.

---

## ⚙️ How It Works

```text
User Request
      │
      ▼
Frontend (React)
      │
      ▼
Local JSON Database
      │
      ▼
Language Selection
      │
      ▼
AI Animation
      │
      ▼
Translated News Display
```

**No external translation API is used during runtime**, ensuring fast response times and reliable performance.
